# my-smart-ai-helper-
## описание проекта
мой  первый сайт с мозгами на Python. Сайт умеет рекомендовать игры по прдепочтению пользовтеля.
## стек
я использовала html,css,python для разроботки данного сервиса. реализован фулстак сайе, с созданием локального сервера и подключением нейросети к сайту. Использованы библиотеки flask,transformers,pipeline,hf_xet.
модель нейросети для генерациии игр - sberbank-ai. нейросеть для понимание и анализа настроение - blanchefort.
## установка библиотек
для установки библиотек вам необходимо ввести следующю строчку кода:
```
pip install -r requirements.txt
```
или установить библиотеку по отдельности:
  pip install flask
  pip install transformes
  pip install pipeline
  pip install torch
  ```
